text-decoration

Définition et utilisation

La text-decoration propriété spécifie la décoration ajoutée au texte et est une propriété abrégée pour :
- ligne-de-décoration-de-texte (obligatoire)
- texte-décoration-couleur
- style de décoration de texte
- texte-décoration-épaisseur

Exemple

p {
text-decoration: overline;
}

exemple

p {
text-decoration: line-through;
}

exemple

p {
text-decoration: underline;
}
