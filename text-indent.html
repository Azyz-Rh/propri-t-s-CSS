text-indent

Définition et utilisation

La text-indent propriété spécifie l'indentation de la première ligne d'un bloc de texte.
Remarque : Les valeurs négatives sont autorisées. La première ligne sera en retrait vers la gauche si la valeur est négative.

Exemple

p {
text-indent: 50px;
}

exemple

p {
text-indent: 2em;
}

exemple

p {
text-indent: 30%;
}
