text-overflow

Définition et utilisation

La text-overflow propriété spécifie comment le contenu débordé qui n'est pas affiché doit être signalé à l'utilisateur. Il peut être tronqué, afficher des points de suspension (...) ou afficher une chaîne personnalisée.
Les deux propriétés suivantes sont requises pour le débordement de texte :
- espace blanc : nowrap ;
- débordement caché;

Exemple

p {
text-overflow: ellipsis;
}
